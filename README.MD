- HTML5
- CSS3
- JavaScript


## Setup

1. Clone the project:  
`git clone https://github.com/coed95/todo-list.git`

2. Install the required dependencies:  
`npm install`

3. Run a live demo:  
`npm start`

4. Run files distribution:  
`npm run build`
